CSS Custom Properties - 7 Custom Properties Control 200+ Paths!

Remember CSS custom properties from the Colors lab? Prof. Teeters taught me I could use them in SVG too! I built my entire Lightbulb around just 7 custom properties. This was the moment everything clicked for me! ü§Ø

HAP's Lightbulb Color System

This is all it takes to control 200+ paths:

:root {
  /* HAP's Lightbulb Color System */
  --bulb-glow: hsl(45, 100%, 89%);      /* Lightest - the emanating light */
  --bulb-warm: hsl(40, 97%, 73%);       /* Medium - warm glass tones */
  --bulb-bright: hsl(34, 100%, 65%);    /* Intense - the hot filament */
  --metal-light: hsl(60, 4%, 74%);      /* Highlight on metal */
  --metal-medium: hsl(60, 2%, 57%);     /* Body of metal base */
  --metal-dark: hsl(0, 1%, 49%);        /* Shadow on metal */
  --base-dark: hsl(210, 4%, 21%);       /* Darkest - grounding element */
}

These 7 custom properties connect to my 7 semantic groups. Each custom property controls dozens of paths! üü†

Prof. Teeters noticed I was still using hex colors from Station 3. "Try HSL format," she suggested. When I converted #ffefcb to hsl(45, 100%, 89%), I could suddenly see that all my warm colors shared similar hues (34-45)! The relationships became visible. HSL made my color system transparent ‚Äî I could see the pattern, not just memorize random hex codes. üü†

Add a :root section at the top of your <style> block:

  --bulb-glow: hsl(45, 100%, 89%);
  --bulb-warm: hsl(40, 97%, 73%);
  --bulb-bright: hsl(34, 100%, 65%);
  --metal-light: hsl(60, 4%, 74%);

  --metal-dark: hsl(0, 1%, 49%);
  --base-dark: hsl(210, 4%, 21%);
}

Notice I'm using hsl() format! Prof. Teeters taught me this makes relationships visible. These are the same 7 groups from Station 3! üü†

Replace the hardcoded hex colors with var():

/* OLD - hardcoded */
.bulb-glow { fill: #ffefcb; }

/* NEW - uses custom properties */
.bulb-glow { fill: var(--bulb-glow); }

Do this for all 7 classes! The classes stay the same, only the value source changes. üü†

Add a theme class that overrides the custom properties:

/* Cool Blue Theme */
svg.theme-cool {
  --bulb-glow: hsl(205, 100%, 95%);
  --bulb-warm: hsl(205, 100%, 82%);
  --bulb-bright: hsl(205, 100%, 64%);
  /* metal colors can stay the same or change - your choice! */
}

To activate: add class="theme-cool" to the <svg> element. That's it! üü†
